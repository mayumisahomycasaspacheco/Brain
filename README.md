# Brain
Brain is an interactive program that allows users to practice multiplication tables in a simple and dynamic way. The user can choose to view the table for any number from 1 to 12, where each result is displayed in order. It also includes a challenge mode in which random multiplication problems are generated using numbers from 1 to 12, and the user must answer each question.

During the challenge mode, the program evaluates in real time whether the answers are correct or incorrect, showing the correct answer in case of an error. At the end, the total number of correct answers is calculated, and the user’s performance is displayed as a percentage, accompanied by a motivational message based on their result.

Finally, the program allows the user to repeat the experience or exit with confirmation, making the interaction more controlled and friendly.

## Concepts used
- Variables (integers and characters)
- Conditional structures (if, else if, else)
- While loop (for the main menu)
- For loop (for the multiplication table and challenge mode)
- Random number generation (rand)
- User input validation
- Basic mathematical operations (multiplication and percentage calculation)
- Program flow control via interactive menu
